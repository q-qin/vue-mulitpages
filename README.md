# 基于vue.js写的多页面webapp [线上访问地址](http://wcjs.3keji.com/)


### 安装

项目地址：（`git clone`）

```shell
git clone https://github.com/q-qin/github.git
```

通过`npm`安装本地服务第三方依赖模块(需要已安装[Node.js](https://nodejs.org/))

```
npm install
```

启动服务(http://localhost:8089)

```
npm run dev
```

发布代码
```
npm run build:test
npm run build
npm run build:all
```
