# 基于vue+webpack+axios写的多页面webapp [线上访问地址](http://wcjs.3keji.com/)

## 骨架屏vue-skeleton-webpack-plugin+dll依赖打包webpack.DllPlugin
``` bash
# 安装依赖
(c)npm install
或者yarn 

# 开启本地服务器localhost:8091
npm run dev
或者yarn dev

# 项目打包(依赖/正式/测试/同时)
npm run dll
npm run build
npm run build:test
npm run build:all

```
